# About

Chat App lets people chat in real-time through your web browser. It has light and dark mode, messages send almost instantly, and it has an amazing UI/UX. Everybody is in the same chat. To use it, make sure Node.js is installed on your computer. You run it locally on your computer.

# Installation

To install it, make sure you have Homebrew installed. If you don't go to www.brew.sh and follow the instructions to download it. Then, run brew install node in your terminal. To start the website, run the terminal command node server.js. Make sure you are in the chat-app folder when running the command. Now, it will be hosted on your computer!
