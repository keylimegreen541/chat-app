# About

Chat App lets people chat in real-time through your web browser. It has light and dark mode, messages send almost instantly, and it has an amazing UI/UX. Everybody is in the same chat. To use it, make sure Node.js is installed on your computer. You host it locally on your computer.

# Installation

You can host Chat App on Windows, Mac, and Linux. To install it, make sure you have Node.js installed on your computer. If you don't have it installed, click [here](https://www.nodejs.org). Now, run this command inside of a terminal window in the chat-app folder: `npm install express socket.io`. To start the website, run the terminal command node server.js. Make sure you are in the chat-app folder when running the command. Now, it will be hosted on your computer on port 3000!
